# 图片目录说明

本目录用于存放项目后续会替换到页面里的本地图片素材。

## 目录分类

- `home/carousel/`
  - 对应首页首屏轮播图。
  - 页面位置：`frontend/src/views/user/Home.vue`
  - 建议文件名：`hero-01.jpg`、`hero-02.jpg`、`hero-03.jpg`

- `products/main/`
  - 对应茶品主图。
  - 页面位置：
    - 首页精选茶品、当季上新、馆主推荐
    - 全部茶品列表
    - 商品详情主图
    - 购物车
    - 收藏页
    - 订单列表
    - 订单详情

- `products/gallery/`
  - 对应商品详情页缩略图、多图展示。
  - 页面位置：`frontend/src/views/user/ProductDetail.vue`
  - 后台位置：`frontend/src/views/admin/AdminProducts.vue`

- `rooms/`
  - 对应包间图片。
  - 页面位置：
    - 包间预约页
    - 我的包间预约
  - 后台位置：`frontend/src/views/admin/AdminRooms.vue`

- `activities/`
  - 对应活动封面图。
  - 页面位置：
    - 茶事活动列表
    - 活动详情
    - 我的报名
  - 后台位置：`frontend/src/views/admin/AdminActivities.vue`
